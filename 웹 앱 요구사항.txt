SDV System Division Agent 요구사항 정의서 (PRD)

1. 프로젝트 개요
    프로젝트 명: SDV System Division Agent

    목적: SDV(Software Defined Vehicle) 시스템 부서의 신규 인원 온보딩 효율화 및 도메인 지식 자산화

    핵심 가치: 신속한 지식 습득, 팀원 성장 가시화, AI 기반의 효율적인 지식 검색 및 검증

2. 사용자 역할 및 계정 (Auth)

    시스템은 권한에 따라 UI 기능과 접근 권한을 동적으로 제어합니다.

    2.1 계정 체계

    - 관리자 (Admin)
    ID: manager / PW: 1234 (테스트용)
    권한: 팀 전체 현황 조회, 지식 베이스 소스 관리, 사용자 권한 부여

    - 사용자 (User)
    ID: user / PW: 1234 (테스트용)
    권한: 개인 온보딩 진행, 지식 베이스 조회 및 AI 질문, 퀴즈 풀기

    2.2 보안 및 데이터 저장

    사용자 계정 및 비밀번호는 암호화된 데이터베이스에 저장되어야 함
    로그인 세션에 따라 좌측 탭 및 기능 버튼이 동적으로 렌더링됨

3. 주요 기능 (Functional Requirements)
    
    3.1 온보딩 및 팀 관리 (Onboarding & Team Mgmt)
    - My Status (사용자용)
    본인에게 할당된 온보딩 항목 리스트 확인
    체크리스트 완료 시 진행도(Progress Bar) 실시간 반영
    
    - Team Management (관리자 전용)
        부서원 전체 리스트 및 개별 온보딩 달성률 모니터링
    특정 팀원의 병목 구간 확인 및 지원 기능

    3.2 지식 베이스 (Knowledge Base)
    - NotebookLM 연동
        공식 링크: https://notebooklm.google.com/notebook/698eab6a-b749-4b74-9dda-786b098d0106
        notebooklm 대화창을 띄움
    - Split View UI
        (좌) 문서 리스트: 업로드된 문서/가이드 목록
        (우) 뷰어: 선택한 문서의 상세 내용 표시
    - 플로팅 AI 챗봇
        스크롤 캡처 또는 텍스트 선택 기반 즉석 질문 기능
        현재 보고 있는 문서의 맥락을 이해하고 답변 제공
    
    
    - 소스 업로드 관리 (Admin 전용)
        NotebookLM에 새로운 소스 업로드 인터페이스 (시뮬레이션 및 외부 링크)
        일반 사용자 중 특정 인원에게 '업로드 권한'을 부여할 수 있는 검색 및 추가 기능
    
    
    3.2.1 Exam Node (학습 검증)
    지식베이스의 서브 기능으로, 학습을 검증합니다.
    - 퀴즈 생성 설정
        지식 베이스 내 특정 문서를 다중 선택하여 퀴즈 범위 지정
        퀴즈 타입 선택 (객관식, OX, 단답형 등)
    - 퀴즈 실행
    문서 내용을 기반으로 생성된 퀴즈 풀이 및 실시간 채점
    오답 시 관련 문서 섹션으로 바로가기 링크 제공
    
    3.4 부가 서비스 (Apps)
    - 회사 인근 맛집 지도: 양재역 등 주요 거점 주변 식당 추천 및 정보
    설문 및 투표: 팀 내 의사결정을 위한 간이 투표 시스템

4. 사용자 인터페이스 (UI/UX)
레이아웃: 좌측 네비게이션 바 + 우측 메인 콘텐츠 영역
디자인 톤: 화이트와 밝은 그레이를 베이스로 한 라이트 모드(Light Mode). 포인트 컬러로 신뢰감을 주는 블루(Soft Blue) 또는 현대적인 민트 계열을 사용하여 깨끗하고 전문적인 느낌 강조
시각 요소: 충분한 여백(Whitespace)과 부드러운 그림자(Soft Shadow)를 활용하여 요소 간 구분 명확화
반응형: 데스크탑 환경에 최적화하되, 맛집 지도 등은 모바일에서도 가독성 확보

5. 향후 확장 계획
Slack/Teams 알림 연동 (온보딩 완료 알림)
개인별 학습 이력 리포트 생성
사내 주요 도구(Jira, Confluence) API 연동
